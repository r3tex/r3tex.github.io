in nihilum

A book, an album, a neuromorphic computing experiment, and the reader. Four substrates for one observation: that experience is substrate-agnostic. Perception comes first; the rest follows.

The whole project traces a hermeneutic loop in four parts:
metaphysics → physics → computation & biology → humanity. Each substrate runs the loop in its own grammar.

// the book

Four parts, sixteen chapters, targeting ~460 pages. Spiritual ancestor: Hofstadter's Gödel, Escher, Bach. Written for the gifted amateur.

// the album — chromaform

Sixteen tracks, one per chapter, grouped into the same four parts. Each song interleaves with its chapter as an interlude. The band is Chromaform.

// the hardware

A neuromorphic experiment run through Nerv Dynamics. Built to test a hypothesis rather than ship a product.

// the reader

The fourth substrate is you. The argument doesn't close on the page, in the speakers, or in the chip — it closes here.
